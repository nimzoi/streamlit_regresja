# Streamlit - regresja liniowa

Aplikacja Streamlit z modelem regresji liniowej (SUML, Lab04).

s25508

## Uruchomienie

```
pip install -r requirements.txt
streamlit run streamlit_app.py
```

## Pliki

- `streamlit_app.py` - aplikacja
- `model_utils.py` - funkcje pomocnicze (trenowanie, przewidywanie)
- `dane.csv` - dane (x, y)
- `model.pkl` - zapisany model (generowany przy pierwszym uruchomieniu)
- `requirements.txt` - zależności
